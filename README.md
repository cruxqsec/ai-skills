# ai-skills

Personal collection of Opencode skills used in my workflows.

Skills compiled from:
- https://github.com/JuliusBrussee/skills
- https://github.com/mattpocock/skills

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/cruxqsec/ai-skills/main/install.sh | bash
```

Installs all skills to `~/.config/opencode/skills/`.
